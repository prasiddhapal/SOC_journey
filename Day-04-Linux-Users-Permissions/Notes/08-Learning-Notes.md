## 💡 Key Concepts

- Every Linux user has a unique UID.
- Every user belongs to one or more groups.
- File ownership and file permissions are different concepts.
- `chmod` changes permissions.
- `chown` changes ownership.
- `umask` controls the default permissions of newly created files.

## 🚀 Skills Practiced

- Modified file permissions using `chmod`.
- Changed file ownership using `chown`.

- Observed how `umask` changes default file permissions.
