# 🛡️ Day 04 – Linux Users, Groups & Permissions

## 📌 Overview

Day 4 focused on Linux user management, file ownership, permissions, and access control. The practical labs covered how Linux identifies users, assigns ownership, controls file access, and applies default permissions using `umask`.

- Learn UID and GID
- Investigate File Ownership
- Analyze File Permissions
- Practice `chmod`
- Practice `chown`
- Understand `umask`
- Apply the Principle of Least Privilege
