# Linux Archive and Compression for SOC Analysts

## 📖 Introduction

Archive and compression tools are widely used in Linux systems to package, transfer, and store files efficiently. In Security Operations Centers (SOC), analysts frequently encounter compressed log files, evidence packages, malware samples, and incident response archives. Understanding how to inspect and handle these files safely is an essential investigation skill.

---

## 🎯 Learning Objectives

- Understand the difference between archiving and compression.
- Create and extract TAR archives.
- Compress and decompress files using GZIP.
- Create and inspect ZIP archives.
- Apply safe archive handling techniques during SOC investigations.

## 🎯 Key Takeaways

- TAR packages multiple files into one archive.
- GZIP compresses files to reduce storage space.
- ZIP archives both packages and compresses files.
- Always inspect untrusted archives before extracting them.
- Evidence should be verified before analysis.
