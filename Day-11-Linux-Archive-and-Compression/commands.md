# Commands Practiced

## TAR

```bash
tar -cvf backup.tar notes.txt commands.txt
```

```bash
tar -tvf backup.tar
```

```bash
tar -xvf backup.tar
```

---

## GZIP

```bash
gzip notes.txt
```

```bash
gunzip notes.txt.gz
```

```bash
gzip -c notes.txt > notes.txt.gz
```

---

## ZIP

```bash
zip backup.zip notes.txt commands.txt
```

```bash
unzip -l backup.zip
```

```bash
unzip backup.zip
```
