# Notes

## Program vs Process

- Program → Stored on disk.
- Process → Running instance of a program.

---

## Investigation Workflow

Alert

↓

top

↓

Identify PID

↓

ps -fp <PID>

↓

Check PPID

↓

Identify Parent Process

↓

Check User

↓

Make Decision

↓

kill / kill -9 (Only if authorized)

---

## Signals

SIGTERM (15)

- Graceful termination

SIGKILL (9)

- Immediate termination

SIGINT (2)

- Generated by Ctrl + C

---

## Important SOC Principle

Never assume.

Investigate.

Collect evidence.

Then conclude.
