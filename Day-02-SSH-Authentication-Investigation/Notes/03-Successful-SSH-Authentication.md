# 🛡️ Successful SSH Authentication

## 🎯 Objective

Generate a successful SSH login and understand how it is recorded in Linux authentication logs.

---

## 📖 Concept

A successful SSH authentication occurs when the user provides valid credentials (password or SSH key). Once authenticated, the SSH server creates a secure session and records the event in the authentication logs.

---

## 🧪 Command

```bash
ssh window@127.0.0.1
```
